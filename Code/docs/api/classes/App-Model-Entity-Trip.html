protected array<string, bool> $_accessible = ['order_number' => true, 'title' => true, 'departure' => true, 'arrival' => true, 'created' => true, 'transport_mode' => true, 'roadtrip_id' => true, 'avoid_highways' => true, 'avoid_tolls' => true, 'departure_time' => true, 'roadtrip' => true, 'sub_steps' => true]

Note that when '' is set to true, this allows all unspecified fields to be mass assigned. For security purposes, it is advised to set '' to false (or remove it), and explicitly make individual fields accessible as needed.
