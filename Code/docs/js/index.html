accessibility.js

Ce script gère les préférences d’accessibilité utilisateur : - Thème sombre / clair - Mode malvoyant (taille/contraste adapté) - Filtres pour différents types de daltonisme Les préférences sont sauvegardées dans le localStorage et réappliquées automatiquement au chargement de la page.

index.js

Ce script initialise une carte Leaflet permettant : - La géolocalisation utilisateur - La recherche d’un lieu via Nominatim - L’affichage de points d’intérêt via l’API Overpass (OpenStreetMap) - Le filtrage par catégorie - Le réglage du rayon de recherche Dépendances : - Leaflet.js - API Overpass - API Nominatim
